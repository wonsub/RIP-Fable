# Lane Specs

Each lane has its own **dedicated goal** (its `/goal`) plus a provably disjoint
file/asset set so lanes never collide. A builder is handed its lane's goal — one
observable sentence — not the whole project goal (D1).

## Format

| Lane | Goal (one observable sentence) | Files/assets (disjoint) | Gate |
|------|--------------------------------|-------------------------|------|
| A    | <what this lane delivers>      | <disjoint set>          | G1   |

(empty — populated per run)
